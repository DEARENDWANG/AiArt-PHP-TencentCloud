# AiArt-PHP-TencentCloud
腾讯云Ai绘画，文生图api请求示例
# 安装 TencentCloud SDK PHP
通过 Composer 安装，执行：<code>composer require tencentcloud/aiart</code>
# 请求示例
<pre>https://example.com/api.php?prompt=提示词&styles=绘画风格&resolution=尺寸</pre>
